# java-blackjack

# 규칙
Blackjack : 처음 두 장의 카드 합 21 => 승   
Bust : 카드 합 21 초과 => 패   
Push : 플레이어, 딜러 카드 합이 같음 => 무승부   
Hit : 플레이어의 카드 2장의 합이 21을 초과하지 않을 경우, 추가 카드를 요청   
Stay : 플레이어가 추가 카드를 원하지 않음, 딜러는 카드 합 17 이상이면 추가 카드를 얻을 수 없음
